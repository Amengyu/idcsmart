阿里云短信认证服务 - 开发者版

这是魔方业务系统 v10 普通短信接口插件，外观和官方短信接口一致，底层调用阿里云号码认证服务 PNVS 的 SendSmsVerifyCode。

服务接口Dypnsapi / SendSmsVerifyCode
签名模板请使用号码认证控制台赠送签名和赠送模板；系统赠送签名必须搭配系统赠送模板。
模板ID插件会读取 config/smsTemplate.php 自动预置模板和 TemplateCode，例如 100001。安装插件自动导入，删除插件自动清理。
变量规则魔方变量 @var(code) 会作为阿里云模板参数 {"code":"验证码","min":"5"} 发送。
校验逻辑当前仍由魔方生成和校验 6 位验证码；有效期固定 5 分钟；阿里云接口负责短信下发。插件不配置验证码长度和有效时长，避免误导。

如果赠送签名变化，请改 config.php 的签名 options；如果赠送模板 CODE 或预填模板变化，请改 config/smsTemplate.php。
